# Student Result Analysis using Pandas

## Project Description
This project analyzes student marks using the Pandas library in Python.

## Features
- Create a DataFrame
- Calculate total marks
- Calculate percentage
- Assign grades (A, B, C)
- Pass/Fail result
- Find topper
- City-wise average percentage
- Grade count
- Student ranking
- Export results to CSV

## Technologies Used
- Python
- Pandas

## How to Run
```bash
pip install pandas
python student_result.py
```

## Output
The program generates:
- Student result table
- Topper details
- City-wise average
- Grade count
- Ranking
- `student_result.csv`
